# ToDo App with Node.js and gRPC ⚡⚡⚡

### Install deps ⬇️:
```bash
$ yarn install
```
### Run the server 🖥️:
```bash
$ node server.js
```

### Add ToDo 📝:
```bash
$ node client.js <ToDo text>
```

### Get all ToDos 📃:
```bash
$ node client.js
```
